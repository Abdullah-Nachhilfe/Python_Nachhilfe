Web-Server

  • Server
    • Software --> bietet einen bestimmten Dienst.
  • Host
    • PC, wo der Server[Programm] läuft.
  • Client
    • Programm --> über ein Netzwerk benutzt diesen bestimmten Dienst.

Client-Server-Schema

Client Server
Sendet Resquest (Anfrage) Response (Antwort)

Web-Server:

  • Server
    • Dieser Server bietet Dienste im World Wide Web (www)
